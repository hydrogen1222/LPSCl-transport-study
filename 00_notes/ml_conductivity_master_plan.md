# 结构到离子电导率机器学习方案

更新日期：2026-04-13

## 1. 先把最终目标说清楚

本项目最后真正想做的不是单纯“算出几个结构”，而是：

**输入一个 Li6PS5Cl 派生结构，预测其输运标签，并尽可能逼近离子电导率。**

但是这个目标必须分层实现。

当前最合理的最终目标不是：

- 输入任意陌生材料结构，直接给出准确实验室温离子电导率

而是：

- 输入 `Li6PS5Cl` 体系内的 bulk / GB / vacancy 结构
- 预测其模拟定义下的输运标签
- 最终输出 `Ea`、`D_tracer(T)` 和 `sigma_NE,upper(300 K)`

这是当前时间和数据规模下可落地、可答辩、可迭代的版本。

## 2. 为什么当前结果不能直接当“准确电导率”

当前已经拿到的 `bulk_ordered`、`gb_Sigma3_t3`、`gb_Sigma3_t3_Li_vac_c1_s1` 三组正式 MD 结果，可以用于趋势分析，但绝对值还不够稳。

原因有四个：

1. 当前用的是 `tracer diffusion` 而不是集体电荷扩散。
2. 当前电导率来自 `Nernst-Einstein` 假设，默认 `Haven ratio = 1`。
3. 当前每个温度点只有单条轨迹，统计量偏弱。
4. 当前是从 `600 / 700 / 800 K` 向 `300 K` 外推，误差会被放大。

因此当前结果的正确定位应是：

- 可用于比较 bulk / GB / vacancy 的相对趋势
- 可作为机器学习标签的第一版候选
- 不能直接冒充实验等价值

## 3. 正确的标签体系

如果要做结构到输运的机器学习，标签不能只有一个。

建议分成三层：

### 第一层：原始动力学标签

- `D_tracer(600 K)`
- `D_tracer(700 K)`
- `D_tracer(800 K)`

这是最直接、最少二次处理的标签。

### 第二层：压缩后的物理标签

- `Ea`
- `ln(D0)`

优点：

- 可以压缩多温度结果
- 更符合 Arrhenius 物理图像
- 便于模型学习

### 第三层：展示用标签

- `sigma_NE,upper(300 K)`

这个量可以保留，但必须明确：

- 它是上界估计
- 它依赖 `Haven ratio = 1`
- 它不等于最终实验值

## 4. 数据集应该长什么样

当前已有结构池已经不错，但标签还不够多。

建议的数据集分为两部分：

### A. 高可信标签核心集

先做稳的结构：

- `bulk_ordered`
- `gb_Sigma3_t3`
- `gb_Sigma3_t3_Li_vac_c1_s1`

要求：

- 每个温度至少两条独立随机种子
- 或每个温度延长到更长轨迹

目的：

- 建立高可信参考标签
- 用来校准后续机器学习模型和误差范围

### B. 中等规模训练集

把已有结构家族系统纳入：

- bulk ordered
- bulk vacancy
- GB `t1 / t2 / t3`
- GB + vacancy

目标规模：

- `20-40` 个结构的第一版训练集

如果时间和算力允许，再扩展到：

- `50-100` 个结构

## 5. 特征工程优先于大模型

以你当前数据规模，最不建议的路线是：

- 直接从零训练大 GNN

更现实的路线是：

### 第一版模型

- `XGBoost`
- `RandomForest`
- `ExtraTrees`

输入特征建议：

- bulk / GB / vacancy 类型
- vacancy 数量
- cell volume
- Li 数密度
- Li-S / Li-Cl 最近邻均值和方差
- Li 局域配位统计
- 自由体积 / Voronoi 指标
- CP2K 单点能、最大力、平均力（可选）

### 第二版模型

如果后续数据量真的上来，再试：

- structure graph
- pretrained embedding
- GNN regression

## 6. 机器学习模型应该预测什么

建议不要直接只预测一个 `sigma(300 K)`。

更稳的模型输出是：

1. `Ea`
2. `ln(D0)`
3. `sigma_NE,upper(300 K)` 作为派生量
4. 不确定性估计

这样做的好处是：

- 模型更接近物理过程
- 后续可以恢复任意温度下的 `D(T)`
- 更容易解释 GB / vacancy 为什么改变输运

## 7. 当前项目阶段应该如何推进

按优先级排序，当前最值得做的是：

1. 统一把现有 conductivity 结果改口为 `sigma_NE,upper`
2. 稳定三条核心 structure 的 MD 标签
3. 扩展一批 `20-40` 个 structure 的第一版训练集
4. 搭建特征表和 baseline 回归模型
5. 只在 baseline 可用后，再考虑更复杂模型

## 8. 毕设中最稳的学术表述

当前最稳的论文叙事是：

**本工作利用 UMA 机器学习势函数生成 Li6PS5Cl 缺陷与晶界结构的动力学标签，并结合少量 CP2K 验证，构建结构到输运标签的初步机器学习预测框架。**

这个说法有三个优点：

- 不夸大为“已经精确预测实验值”
- 保留机器学习主线
- 和你后续继续做预测器自然衔接

## 9. 当前不应该做的事

不建议现在走这三条路：

1. 为了让 `15.6 mS/cm` 看起来像实验值而强行圆回来
2. 立即重训新的势函数
3. 在样本只有个位数时直接宣称“输入任意结构即可预测电导率”

这三条都会让论文变虚。

## 10. 最终建议

你现在真正应该接受的工程现实是：

- 当前结果已经足够开始做“结构到输运标签”的机器学习路线
- 但当前标签必须被重新定义为模拟定义下的上界或代理量

只要这一步定义清楚，整个项目仍然是连贯的：

- 结构生成
- UMA-MD 打标签
- CP2K 做局部校验
- 用机器学习回归输运标签

这就是当前最稳、最能落地的毕业设计主线。
