# 当前电导率结果的准确性说明

更新日期：2026-04-13

## 1. 结论先行

当前 `conductivity_production_summary.csv` 里的室温电导率不能被解释为最终准确的实验可比值。

当前更合理的定义是：

**`sigma_NE,upper(300 K)`，即基于 tracer diffusion 和 Nernst-Einstein 关系得到的上界估计。**

## 2. 为什么会偏高

当前结果偏高并不意外，主要原因是：

1. 使用了 tracer diffusion，而不是集体电荷扩散。
2. Haven ratio 默认为 1，没有进行相关扩散修正。
3. 使用总 Li 数密度换算电导率，因此更接近理论上界。
4. 目前每个温度点只有一条轨迹，统计量有限。
5. 从高温点外推到室温会放大误差。

## 3. 现在这些结果还能不能用

能用，但用途应限于：

- bulk / GB / GB + vacancy 之间的相对趋势比较
- Arrhenius 斜率与 `Ea` 的第一版估计
- 后续机器学习标签的初始版本

不应直接用于：

- 宣称已经准确再现实验室温离子电导率
- 对绝对数值做强结论

## 4. 最稳的写法

后续文档和论文中推荐统一写成：

- `D_tracer(T)`
- `Ea`
- `sigma_NE,upper(300 K)`

不再把当前结果简称为“室温离子电导率”。

## 5. 如果想继续提高可信度

最值得补的不是重开所有流程，而是：

1. 对三个核心结构每个温度补一条独立随机种子。
2. 或将现有轨迹延长到 `50000` 步以上。
3. 或在后处理中加入相关扩散修正。

这三条的投入产出比都比盲目扩大结构数更高。
